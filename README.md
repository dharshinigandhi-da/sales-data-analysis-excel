# Sales Data Analysis using Excel

## Objective
To analyze sales data and identify basic KPIs and trends using Microsoft Excel.

## Tools Used
- Microsoft Excel

## KPIs
- Total Sales
- Total Quantity Sold
- Average Sales

## Analysis
- Data cleaning and validation
- Monthly sales trend using pivot tables
- Product-wise sales analysis
- Visualization using line and bar charts

## Insights
- March recorded the highest sales
- Bags generated the highest revenue
- Sales showed an increasing monthly trend
